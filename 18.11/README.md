## Сдвиг курсора

При работе с файлом есть 2 курсора: на чтение и на запись.

- `seekg` - seek get для чтения данных
- `seekp` - seek put для записи данных

Также есть флаги состояния файла: `file.eof()`, `file.fail()` и т.д.
